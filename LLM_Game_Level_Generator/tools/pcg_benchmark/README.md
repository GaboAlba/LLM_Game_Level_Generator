# PCG Benchmark Runner

This directory contains scripts for running the PCG (Procedural Content Generation) benchmark tests on levels generated by the LLM Game Level Generator.

## Prerequisites

- **Python 3.8+** installed and available in PATH
- **PowerShell 5.1+** (Windows) or PowerShell Core (cross-platform)
- Internet connection (for installing dependencies)

## Files

- **`run.ps1`**: PowerShell script that orchestrates the entire benchmark workflow
- **`LLM_Generator_Benchmark.py`**: Python script that runs individual benchmark tests
- **`requirements.txt`**: Python dependencies (pcg_benchmark library)
- **`benchmark_results/`**: Output directory for benchmark results (created automatically)

## Quick Start

Run the benchmark with default settings (10 iterations):

```powershell
.\run.ps1
```

### Custom Iterations

Run with a specific number of iterations:

```powershell
.\run.ps1 -Iterations 25
```

### Custom Output Directory

Specify a different output directory:

```powershell
.\run.ps1 -OutputDir "C:\benchmarks\output"
```

### All Parameters

```powershell
.\run.ps1 -Iterations 50 -OutputDir ".\results" -VenvPath ".\my_venv"
```

## What the Script Does

1. **Creates Python Virtual Environment**: Isolates dependencies in `.\venv`
2. **Installs Dependencies**: Installs `pcg_benchmark` from GitHub
3. **Runs Benchmark Iterations**: Executes `LLM_Generator_Benchmark.py` N times
4. **Saves Results**: Outputs JSON files to `benchmark_results/` directory
5. **Cleanup**: Removes the virtual environment after completion

## Output Format

Each benchmark run produces a JSON file with the following structure:

```json
{
  "iteration": 1,
  "timestamp": "2026-02-14T15:30:45.123456",
  "metrics": {
    "playability_score": 0.85,
    "difficulty_score": 0.72,
    "diversity_score": 0.91,
    "overall_score": 0.83
  },
  "duration_seconds": 12.34,
  "errors": [],
  "warnings": []
}
```

## Troubleshooting

### Python Not Found

Ensure Python is installed and in your PATH:

```powershell
python --version
```

### Permission Errors

If you get execution policy errors, run:

```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
```

### pcg_benchmark Installation Fails

Try installing manually:

```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
pip install git+https://github.com/amidos2006/pcg_benchmark.git
```
